# Depression Symptoms Test

Simple console Python app for screening depressive symptoms (inspired by Beck Depression Inventory - BDI).

## Features
- 21 questions
- Input validation (only 0-3)
- Score calculation & interpretation
- Keeps console open after results

python depression_test.py
